# 🏷️ Labels du projet Festival Scout IA

> Si vous ne pouvez pas créer les labels directement, copiez-collez ces informations dans l'interface GitHub.

---

## 📋 Procédure pour créer les labels

1. Allez sur : `https://github.com/nassimus13/Scout-Festival/labels`
2. Cliquez **"New label"**
3. Remplissez :
   - **Label name** : (voir ci-dessous)
   - **Description** : (voir ci-dessous)
   - **Color** : (voir code couleur)
4. Cliquez **"Create label"**
5. Répétez pour tous les labels

## 💡 Combinaisons recommandées

- `Prompts IA` + `High` + `À valider`

- `Bug / correction` + `Urgent` + [domaine spécifique]

- `Recherche festivals` + `Medium` + `Make / n8n`

- `Documentation` + `Low` + `À valider`
